# 功能模块：问卷系统

## 参观前问卷 `/survey/pre`

### 步骤
1. 基本信息：姓名、邮箱、性别、参观日期、城市/国家、首次参观？
2. 艺术背景：初次接触 / 略知一二 / 有所了解 / 专业背景
3. 感兴趣方向（多选）：欧洲建筑与雕塑、中世纪艺术、文艺复兴、巴洛克、新古典主义、印象派、欧洲与新世界
4. 体验方式：故事叙述 / 系统知识 / 学术深度 / 轻松随性
5. 特别期望（开放文字）

### 触发逻辑
- 提交后生成 `profileTag`（用户画像标签）存入 KV
- 若填写邮箱 → 自动发送问卷回顾邮件 + Referral 码
- 管理员收到 Formspree 通知

---

## 参观后问卷 `/survey/post`

### 步骤
1. 基本信息 + 星级评分（整体体验、讲解清晰度、导览节奏）
2. 最深刻的部分（多选）：背景与历史、中世纪艺术、文艺复兴（拉斐尔特展）、十七至十八世纪、印象派、其他
3. 价格感知（每人）：$80–100 / $100–120 / $120–150 / $150+
4. NPS 推荐指数（0–10）
5. 评语 + 公开授权 + 未来兴趣 + 邮箱

### 公开授权（2026-05-06 新增）
> 我们可以匿名公开您的评语，用于向未来的访客介绍导览体验吗？
- **可以，欢迎公开** → `allowPublic: true`
- **不，请保密** → `allowPublic: false`
- 未作答 → `null`
- 管理员邮件显示：✅ / ❌ / —

### 触发逻辑
- 若填写邮箱 → 自动发送问卷回顾邮件 + Referral 码
- 管理员收到 Formspree 通知（含公开授权状态）

---

## Referral 推荐码
- 提交问卷时调用 `createReferral(email, name)`
- 生成唯一码存入 KV
- 邮件中展示推荐码，朋友使用后双方各享优惠
