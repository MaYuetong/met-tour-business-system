# 功能模块：预约系统

## 用户流程
1. 用户访问 `/book`
2. 填写：姓名、邮箱、电话、参观日期（最早 2026-05-09）、人数、套餐
3. 提交 → API `/api/bookings` → 存入 KV
4. 自动发送确认邮件给用户（含预约码、导览员微信、导览详情）
5. 管理员收到 Formspree 即时通知邮件

## 预约码格式
`YYYYMMDD` + 3 位随机大写字母，例：`202505121AB`

## 讲解员匹配

## 核心文件
- `app/book/page.tsx` — 预约表单（客户端）
- `app/book/success/page.tsx` — 成功页
- `app/api/bookings/route.ts` — POST 处理
- `lib/db.ts → addBooking()` — KV 写入
- `lib/email.ts → sendBookingConfirmation()` — 邮件发送
