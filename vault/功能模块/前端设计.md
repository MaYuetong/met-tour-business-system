# 功能模块：前端设计

## 设计理念
- **参考**：大都会博物馆官网 + Spotify UI
- **原则**：扁平化、无阴影、克制留白、小号标题
- **字体**：Noto Serif SC（正文/标题）+ Noto Sans SC（UI/导航）

## 色彩系统

| 变量 | 值 | 用途 |
|------|----|------|
| `--met-red` | `#E51B23` | 主色、CTA、选中状态 |
| `--met-red-dark` | `#C01018` | Hover 加深 |
| `--met-bg` | `#FFFFFF` | 页面背景 |
| `--met-text` | `#1A1A1A` | 主要文字 |
| `--met-muted` | `#444444` | 辅助文字、标签 |
| `--met-border` | `#E5E5E5` | 边框、分隔线 |

> ⚠️ 旧版暖米色 `#F8F5F0` 和酒红 `#A6192E` 已全部替换

## 无阴影原则
全站所有 `shadow-*` Tailwind 类已移除（2026-05-06），包括：
- `shadow-sm` `shadow-xl` `shadow-2xl`
- 所有 `shadow-[...]` 自定义值
- 替代方案：用 `border` 区分层级

## 首页结构
1. AnnouncementBar（顶部公告栏，红底）
2. 导航
3. Hero — 中英双语，无背景图纹理
4. 滚动字幕 Marquee（中英交替）
5. 导览特色 / 行程 / 定价
6. 底部 Footer（中英双语）
