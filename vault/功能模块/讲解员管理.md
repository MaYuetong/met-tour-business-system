# 功能模块：讲解员管理

## 数据结构
```typescript
type GuideRecord = {
  id: string;
  name: string;
  wechatId: string;
  email?: string;
  isActive: boolean;
  availableDates: string[];  // YYYY-MM-DD
  createdAt: string;
};
```

## 匹配逻辑
```
findGuideForDate(date?)
  ↓
过滤 isActive=true 的讲解员
  ↓
有 date → 找 availableDates.includes(date) 的讲解员
  ↓ 未找到
返回第一位活跃讲解员（fallback）
  ↓ 无活跃讲解员
返回 null → 邮件 fallback 到 "Yuti_9999"
```

## 管理后台
- 路径：`/admin/guides`
- 功能：添加讲解员、设置可工作日期、切换启用状态、删除

## 扩展设想
- 未来添加学生讲解员时，为其设置可工作的具体日期
- 系统自动将对应日期的预约分配给该讲解员
- Yuti 作为 fallback 兜底所有未匹配的预约
